## Swapnil Mallick's Academic Homepage

Source code for my personal academic website: [swapnilmallick.github.io](https://swapnilmallick.github.io/)

### About

This site serves as my academic homepage, featuring my research experience, publications, academic projects, coursework, skills, and certifications. It is built on the template by [Jon Barron](https://github.com/jonbarron/jonbarron_website), with visual enhancements to the layout and styling.

### Structure

| Path | Description |
|------|-------------|
| `index.html` | Main page |
| `stylesheet.css` | Site-wide styles |
| `images/` | Logos and profile photo |
| `papers/` | PDFs of publications, project reports, and certificates |
| `blogs/` | Blog post pages |
